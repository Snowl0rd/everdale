# Эвердейл — Сайт кампейна

## Структура файлов

```
everdale/
├── index.html          ← Главная
├── world.html          ← Мир + интерактивная карта
├── npcs.html           ← НПС
├── pantheon.html       ← Боги
├── data/
│   ├── npcs.json       ← Все НПС (редактируй здесь)
│   ├── cities.json     ← Города (редактируй здесь)
│   └── gods.json       ← Боги (редактируй здесь)
├── images/
│   ├── map.jpg         ← Карта Эвердейла (помести сюда)
│   ├── npcs/           ← Картинки НПС (имя файла = image в JSON)
│   ├── cities/         ← Картинки городов
│   └── gods/           ← Картинки богов
├── css/
│   └── style.css
└── js/
    ├── nav.js
    ├── npcs.js
    ├── world.js
    └── pantheon.js
```

## Как добавить нового НПС

Открой `data/npcs.json` и добавь в конец массива новый блок:

```json
{
  "id": "unique-id",
  "name": "Имя персонажа",
  "race": "Раса",
  "role": "Роль или занятие",
  "image": "images/npcs/filename.jpg",
  "description": "Описание персонажа.",
  "tags": ["дружелюбный"]
}
```

Доступные теги: `дружелюбный`, `опасный`, `подозрительный` — влияют на цвет плашки.
Можно добавлять свои — они появятся как серые плашки.

Картинку положи в папку `images/npcs/` и укажи путь в поле `image`.
Если картинки нет — оставь поле пустым `"image": ""`, появится заглушка.

## Как добавить новый город

Открой `data/cities.json` и добавь блок:

```json
{
  "id": "city-id",
  "name": "Название",
  "subtitle": "Короткий эпитет",
  "image": "images/cities/filename.jpg",
  "mapX": 50,
  "mapY": 50,
  "description": "Краткое описание (видно на карточке).",
  "details": "Подробное описание (видно в модале).",
  "pointsOfInterest": [
    { "name": "Название места", "desc": "Описание." }
  ],
  "atmosphere": "Прилагательное, прилагательное"
}
```

`mapX` и `mapY` — координаты маркера на карте в процентах (0–100).
0,0 = левый нижний угол. 100,100 = правый верхний.

## Как добавить нового бога

Открой `data/gods.json` и добавь блок:

```json
{
  "id": "god-id",
  "name": "Имя",
  "domain": "Домен, Домен",
  "alignment": "Благой",
  "symbol": "Описание символа",
  "image": "images/gods/filename.jpg",
  "description": "Описание бога.",
  "worship": "Кто поклоняется"
}
```

Значения alignment для фильтров: `Благой`, `Нейтральный`, `Злой`.

## Как добавить карту Эвердейла

1. Положи изображение карты в `images/map.jpg`
2. Открой `js/world.js`
3. Найди строки `const imgW = 2000;` и `const imgH = 1400;`
4. Замени на реальные размеры твоей карты в пикселях

После этого маркеры городов встанут на нужные места.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например `everdale`)
2. Загрузи все файлы в репозиторий
3. Зайди в Settings → Pages → Source: `main` / `root`
4. Сайт будет доступен по адресу `https://твой-ник.github.io/everdale`

Для обновления контента — редактируй JSON файлы прямо на GitHub через браузер.
Изменения появятся на сайте через 1–2 минуты.

---

## Важно: CORS при локальном запуске

Если открываешь файлы локально (не через сервер), браузер заблокирует загрузку JSON.
Решение: используй расширение Live Server в VS Code, или просто деплой на GitHub Pages.
